Class: ElectricityPricing

ElectricityPricing(initialPrice, costThreshold, priceHistory)

new ElectricityPricing(initialPrice, costThreshold, priceHistory)

Monitors electricity price fluctuations and triggers alerts if thresholds are exceeded. Implements: - **Encapsulation**: Uses private fields for data protection.
Parameters:
Name Type Description
initialPrice number The initial electricity price.
costThreshold number The maximum acceptable price before triggering an alert.
priceHistory Array.<number> The history of electricity prices.
